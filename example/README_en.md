# polaris_radar Example

This example demonstrates the main features of the polaris_radar chart library.

## Features

- **Radar chart** — 6-axis attribute radar chart (Strength, Agility, Intelligence, Stamina, Defense, Magic)
- **Multi-dataset overlay** — Three datasets displayed simultaneously (Character A, Character B, Average) with distinct colors and marker shapes
- **Data switching** — Toggle between two data sets via the top-right button with smooth transition animation
- **Legend widget** — Built-in `RadarLegendItem` displayed on the right side

## How to run

```bash
cd example
flutter run -d chrome
```

Or other supported platforms:

```bash
flutter run -d macos
flutter run -d ios
```
