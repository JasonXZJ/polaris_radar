# polaris_radar 示例

本示例展示了 polaris_radar 雷达图插件的主要功能。

## 功能展示

- **雷达图展示** — 6 轴属性雷达图（力量、敏捷、智力、耐力、防御、魔力）
- **多数据集叠加** — 同时显示三个数据集（角色 A、角色 B、平均值），不同颜色与标记形状
- **数据切换** — 点击右上角按钮可在两组数据间切换，伴随平滑过渡动画
- **图例组件** — 右侧显示内置的 `RadarLegendItem` 图例

## 运行方式

```bash
cd example
flutter run -d chrome
```

或其他支持的平台：

```bash
flutter run -d macos
flutter run -d ios
```
