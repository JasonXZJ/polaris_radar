## 0.1.0

- Initial release.
- `PolarisRadarChart` widget with implicit animations.
- `PolarisRadarData` and `PolarisDataSet` data models.
- Per-dataset dashed/solid borders via `RadarLineStyle.dashArray`.
- Vertex marker shapes: `circle`, `square`, `diamond`, `triangle`.
- Circular and polygon grid via `RadarGridShape`.
- `RadarLegendItem` widget.
